# mesto-react

---

### Обзор

Mesto - версия приложения [mesto](https://github.com/Taashev/mesto-react.git) переписанное под свой сервер.

### Технологии при разработке:

- БЭМ (Nested)
- HTML 5 (Semantic)
- Flex-box
- Grid layout
- Position
- @media
- @import
- JavaScript (ES6)
- OOP
- API
- React 18.1 (CRA)
